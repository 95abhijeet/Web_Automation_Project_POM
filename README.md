## üöÄ **Project Overview**

This automation script performs the following tasks:

- Scrolls to the input form  
- Fills text fields (name, email, phone)  
- Selects radio buttons and checkboxes  
- Selects a country from a dropdown  
- Enters dates into date pickers  
- Clicks the submit button using a JavaScript executor  
- Handles any alert pop-ups  
- Takes a screenshot after form submission
